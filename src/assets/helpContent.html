Audio monitoring:

  • Connects different devices of a wirless network (PC or mobiles) to monitor audio
  • Download the PC version and walk through videos at: http://airBabyCall.com

    • Requirements:

  • For PC or Mac: Network firewall off
  • At least two devices (mobiles or PCs) connected to the same network running the app
  • The devices should be within the same network, and not tethering the wifi to each others
  • Connect devices to their power cable (monitoring audio demands battery)

    • Start Monitoring:

  • Start Air-BabyCall app in two devices (ore more)
  • When the text message in right hand top reads: 'Set a Baby Unit', choose a device to be a Baby Unit by hitting the button "set as Baby unit"
  • To swap the units betweeb Baby unit and Parent unit hit 'Set a Baby Unit' in the device you wish to be the Baby unit

    • Messages:

  • "Disconnected. Run at least one more app!"
    • This message is displayed until the devices running the app find each others.
    See Troubleshooting if the message doesn't go away over a minute or so.
  • "Set a baby unit"
    • As soon as devuces running the app find each others this messages will appear. Hit the button "Set as Baby unit" in one of the devices to start monitoring
  • "Parent Unit in background mode?" OR "Baby Unit in background mode?"
    • a) Check whether the unit of the message did not run out battery (always connect units to power charger) b) Blow air onto the Baby unit microphone for at least five seconds and check if sound comes in the Parent unit, or if there is a Baby crying animation
    If no sound or animation start it again (see Start Monitoring)

    Troubleshooting:< *If using PC or Mac version, ensure there is no netowrk firewall on
    **If you are using an iPad or iPhone "Set a Baby Unit" before the device goes to background mode

  • Disconnect the devices from the wireless and then connect them again
  • Make sure the devices are connected to the same wireless network
  • Kill the Air BabyCall in both devices, and then start them again
  • Within a minute or so the message in the top hand side: "Disconnected" should be replaced by "Set a Baby Unit"
  • If you still didn't manage it send us an email to support AT airbabycall.com
