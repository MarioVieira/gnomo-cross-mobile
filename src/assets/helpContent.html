Requirements:

  1. For PC or Mac: Network firewall off
  2. At least two devices (mobiles or PCs) connected to the same network with BabyCall app
  3. The devices shoould be within the same network, and not tethering the wifi to each others
  4. Connect devices to their power cable (monitoring audio demands battery)

Start Monitoring:

  1. Start Air-BabyCall app in two devices (ore more)
  2. When the text message in right hand top reads: 'Set a Baby Unit', choose a device to be a Baby Unit by hitting the button "set as Baby unit"
  3. To swap the units betweeb Baby unit and Parent unit hit 'Set a Baby Unit' in the device you wish to be the Baby unit

Troubleshooting:
  1. If using PC or Mac version, ensure there is no netowrk firewall on
  2. Disconnect the devices from the wireless and then connect them again
  3. Make sure the devices are connected to the same wireless network
  4. Kill the Air BabyCall in both devices, and then start them again
  5. The message in the top hand side: "Disconnected" should be replaced by "Set a Baby Unit"
